## Problems?

**Page won't load?**

- Wait a few more minutes
- Refresh the page
- Check that both services show "Running" in Render dashboard

**Language doesn't switch?**

- Hard refresh: Ctrl+Shift+Delete
- Clear all cache
- Try again

**Can't find the Deploy button?**

- Make sure you're logged in with GitHub
- Make sure you're on dashboard.render.com (not render.me or other site)
- Close any adblockers
