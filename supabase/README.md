This folder is intentionally present to satisfy GitHub/Supabase repository integration.

If you plan to use the Supabase CLI, run `supabase init` locally to populate this folder with configuration, functions, and migrations.

To install the Supabase CLI on Windows (PowerShell):

```powershell
npm install -g supabase
# or using scoop/chocolatey if you prefer
# scoop install supabase
# choco install supabase
```

After installing the CLI, run `supabase init` from the repository root.

Remove this placeholder once you have real Supabase config files in place.
